# PhonIQ

**Know every call. Track every rupee.**

Offline-first Android app: **Phone** (dialer + call IQ), **Messages** (SMS/RCS smart inbox), **Money** (spend & budgets from parsed bank/UPI SMS on device).

- **Min OS:** Android **15** (API **35**); validated on **15 & 16**
- **Stack:** Kotlin, Jetpack Compose, Room — see `android/`
- **Design mockup:** `design/phoniq-mockup-v1.html`
- **Product spec:** `docs/PROJECT.md`

**Privacy:** no ads, no telemetry, **no `INTERNET`** in the app manifest; Google Drive only for **manual** backup (no background sync).

> Early development — APIs and structure will evolve. Issues and PRs welcome after a CONTRIBUTING note is added.
