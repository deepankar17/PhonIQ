Phase 1 scaffold: Room DB, SMS parser engine, repositories, ViewModels

Build system:
- Add KSP 2.1.10-1.0.31 + Room 2.7.1 + ViewModel 2.10.0 + coroutines to Gradle catalog
- Hilt skipped (AGP 9.2 incompatibility with BaseExtension API; tracked for Phase 2)
- android.disallowKotlinSourceSets=false for KSP source registration under built-in Kotlin
- Application container pattern (lazy singletons in PhonIQApp) replaces DI framework

Permissions: READ/WRITE_SMS, READ/WRITE_CONTACTS, READ/WRITE_CALL_LOG,
  RECORD_AUDIO, READ_PHONE_STATE, ANSWER_PHONE_CALLS, FOREGROUND_SERVICE,
  POST_NOTIFICATIONS added to AndroidManifest.xml

Room database (v1 — 9 tables, exportSchema=false):
  contacts, call_log, sms_messages, transactions, accounts,
  budgets, spam_numbers, categories, otp_log
- Entities with FK constraints, indices, and typed enums (type strings)
- DAOs: ContactDao, CallLogDao, SmsDao, TransactionDao, AccountDao,
  BudgetDao, SpamNumberDao, OtpLogDao — all Flow-based
- PhonIQDatabase (abstract RoomDatabase)

SMS parser engine (100% offline, no INTERNET):
- SmsParser: classifies sender+body into OTP/TRANSACTION/SPAM/PROMO/
  BILL/DELIVERY/TRAVEL/PERSONAL/OTHER
- OTP extraction with 5 regex patterns + TTL inference (min/hr/sec)
- Transaction extraction: INR amount regex, debit/credit keyword sets,
  merchant pattern matching, account last4 extraction
- Category inference: 10 categories from keyword sets
  (FOOD/SHOPPING/TRANSPORT/BILLS/EMI/SALARY/INVESTMENT/HEALTH/ENTERTAINMENT/ATM)

Repositories (plain Kotlin, not Hilt-injected):
- SmsRepository: observeAll/OTPs/Transactions, syncDeviceSms (ContentProvider),
  markAllRead, search
- CallLogRepository: observeAll, syncDeviceCallLog, markSpam/unmarkSpam, clearAll
- ContactsRepository: observeAll, search, syncDeviceContacts (ContentProvider)
- TransactionRepository: period queries, totalSpent, budget CRUD, currentMonthEpochRange

ViewModels (with nested Factory for ViewModelProvider):
- MessagesViewModel: unreadCount, allMessages, otpMessages, searchResults (debounced)
- PhoneViewModel: allCalls, allContacts, filteredContacts, syncFromDevice, markSpam
- MoneyViewModel: currentMonthTransactions, totalSpentThisMonth, budgets
