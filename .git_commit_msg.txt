Phase 1: permission gate, ViewModel wiring, dialpad, in-call screen, SMS receiver

Permission flow:
- PermissionScreen: full-screen rationale + Grant button (READ_SMS, READ_CALL_LOG,
  READ_CONTACTS, READ_PHONE_STATE, CALL_PHONE + optional RECORD_AUDIO, POST_NOTIFICATIONS)
- PermissionBanner: compact inline banner when user skips (shows above tab bar)
- allCorePermissionsGranted() check on every app start
- PhonIQRoot: permission gate → shell flow with rememberSaveable state

ViewModel wiring in MainActivity:
- ViewModels created via Factory from PhonIQApp container (smsRepository, callLogRepository,
  contactsRepository, transactionRepository)
- LaunchedEffect triggers syncFromDevice() when permissions become granted
- Real Room data flows via collectAsState(); falls back to SampleData when DB is empty
- MessagesScreen + PhoneScreen signatures updated from SnapshotStateList to List<T>

Entity mappers (data/mapper/EntityMappers.kt):
- CallLogEntity.toRecentCall(): type→CallDirection, relative timestamp, duration caption,
  international number detection, spam/blocked flags
- SmsMessageEntity.toMessageThread(): category→Set<MessageThreadCategory>, pill labels,
  relative timestamp, unread flag
- List<SmsMessageEntity>.toMessageThreads(): group by threadId, latest-message snippet,
  sort by timestamp

Dialpad (DialpadSheet.kt):
- Full 3x4 key grid with digit + letter sub-labels
- Animated call button (green when digits entered, disabled when empty)
- Backspace support (long-clickable display row)
- Launches ACTION_CALL intent; CALL_PHONE permission added to manifest
- Wired to PhoneScreen FAB on Recent tab (replaces toast stub)

InCallScreen (InCallScreen.kt):
- Pulsing circular avatar with initials
- Live call timer via LaunchedEffect coroutine (h:mm:ss / m:ss)
- Mute / Speaker / Keypad in-call action buttons with active-state tint
- Red end-call button (80dp circle)
- Keypad sub-view (switches DialpadContent inline)

SMS BroadcastReceiver (SmsReceiver.kt):
- Registered for android.provider.Telephony.SMS_RECEIVED, priority 999
- Parses each incoming PDU through SmsParser (OTP/TXN/SPAM/…)
- Inserts SmsMessageEntity into Room on IO dispatcher
- Registered in AndroidManifest with BROADCAST_SMS permission gate
